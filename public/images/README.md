# Images Directory

Place your images in this directory with the following names:

- `avatar-woman.jpg` - Profile image of a young woman (or .png, .webp)
- `avatar-man.jpg` - Profile image of a young man (or .png, .webp)
- `avatar-person.jpg` - Profile image of a person (or .png, .webp)
- `background.jpg` - Background image for the hero section (or .png, .webp)

## Image Requirements

- **Avatar images**: Should be square, minimum 64x64px, recommended 128x128px or larger
- **Background image**: Recommended size 1920x1080px or larger for best quality

## Fallback Behavior

If local images are not found, the app will automatically fall back to the original Google user content URLs. This ensures the site continues to work even if local images haven't been uploaded yet.

## Supported Formats

- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)
